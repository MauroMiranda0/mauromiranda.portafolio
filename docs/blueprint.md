# **App Name**: Mauricio Vergara Portfolio

## Core Features:

- Profile Showcase: Display of professional title, expertise and value proposition.
- About Me Section: Detailed personal information and background.
- Skills Showcase: Highlights technological skills and tool expertise.
- Portfolio Display: Showcase featured projects with technical descriptions.
- Contact: Display contact and location details

## Style Guidelines:

- #004F6E
- #F5F5F5
- #00AEEF
- #F4B400
- #1C1C1C
- Font pairing: 'Space Grotesk' sans-serif for headers, and 'Inter' sans-serif for body text.
- Clean and professional icons for skills and tools.
- Minimalist design with clear sections.
- Subtle transitions when navigating the site.
